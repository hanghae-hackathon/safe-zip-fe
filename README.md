<h1>Safe Zip</h1>

## 개요

- 전세 사기 문제 해결을 위한 AI 서비스

## 문제 제기

- 끊임없이 발생하는 전세 사기
- 임차인과 임대인 사이에 커지는 불신

## 해결 방안

- 전세 계약 시 여러 문서 및 지표를 검토해주고 조언해주는 AI 서비스를 개발하자
- 임대인의 자발적인 검증을 통해 임대/임차인간의 신뢰를 쌓을 수 있게 해주는 AI 서비스를 개발하자
